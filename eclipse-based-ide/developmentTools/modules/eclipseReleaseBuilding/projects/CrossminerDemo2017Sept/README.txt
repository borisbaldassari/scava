===============
Library Update:
===============
// Kicserél egy library-t és hozzáigazítja a kódot

1. Kérd a helyimenübõl a "Referenced Libraries > carolinedb.jar"-on / az eszköztárból "CROSSMINER > Update Libraries"
// A "Demo.java" fájlban lesz látható a csere

2. Pipáld be a "[ ] Caroline DB Access" bejegyzést az ablak bal felén

3. Nyomd meg a "List Alternatives" gombot
// Jobb oldalt megjelenik az alternatív library ("Balloon Data Fetcher")

4. Pipáld be az alternatív library-t

5. Nyomd meg az "UpdateSelectedLibraries" gombot
// Megnyílik a fájl, ahol változások lesznek. A változások helyei kiemelõdnek (Package Explorer, Editor)
// Megnyílik a recommendation-öket tartalmazó ablak

6. Nyomd meg az "Apply Recommendations" gombot a recommendation-öket tartalmazó ablakban
// A változások érvénybe lépnek

Recommendations for arbitrary code snippets:
============================================
// Kicseréli a bubble sort-ot insertion sort-tra
